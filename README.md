### JWT를 왜 사용하는가 ?

#### JWT의 사용이유
1. 모바일 앱
2. 모바일 앱에서의 로그아웃
3. 장시간 로그인과 세션

#### API 서버에서 Refresh 토큰을 저장하여 어느 정도의 상태(STATE)를 만드는 이유
JWT의 목적은 로그아웃, Refresh 토큰에 대한 서버 측 제어권한을 만들기 위해서는 Redis와 같은 세션서버를 만들던지, DB에 Refresh 토큰을 조회하는 로직을 만드는 것이 올바르다.
